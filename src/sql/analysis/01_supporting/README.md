# Supporting Analysis Queries

This folder contains the full set of analysis queries (activation → consistency → funnel → retention) developed during exploration.

## How to read
- **Start here**: `../00_story_core/` (queries used directly in `docs/results/story.md`)
- This folder: intermediate / exploratory / validation queries that support the final story

## Folder map
- `01_activation/` : activation definition & base checks  
- `02_consistency/` : consistency metrics vs outcomes  
- `03_funnel_dropoff/` : bottleneck detection (reach/strict, w14/w30)  
- `04_segment_deepdive/` : deeper cuts for key segments and lifts  
- `05_retention/` : cohort & point retention analyses  
- `side_experiments/` : experiments not used in the final story
